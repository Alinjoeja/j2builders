2

Install Python

Skip this step if you already have Python 3.10 or later installed.

1
Go to python.org/downloads
Click the yellow "Download Python 3.x.x" button. This downloads the Windows installer.
2
Run the installer — tick "Add Python to PATH"
This is critical. On the first screen of the installer, check the box that says "Add Python to PATH" before clicking Install Now.
3
Verify the installation
Open Command Prompt (press Win + R, type cmd, press Enter) and run:
python --version
# You should see something like: Python 3.12.3
✅
If you see a version number, Python is ready. Move to Step 3.
3

Install Claude Desktop

1
Download Claude Desktop
Go to claude.ai/download and download the Windows version. It's free.
2
Install and sign in
Run the installer, then sign in with your Claude account (or create a free one at claude.ai).
3
Enable Developer Mode in Claude Desktop
Open Claude Desktop → click your profile icon (top right) → Settings → Developer → toggle on "Enable MCP" if shown. Some versions enable it automatically.
4

Run the Installer

The installer handles everything automatically — Python packages, Claude Desktop config, and path detection. No manual editing required.

1
Extract the downloaded ZIP file
Right-click the ZIP you received after purchase → Extract All → choose any location, e.g. C:\autocad-mcp\
2
Double-click INSTALL.bat
Inside the extracted folder, find INSTALL.bat and double-click it. If Windows asks "Do you want to allow this app to make changes?" — click Yes.
3
Wait for it to finish
A black window will appear and run through the steps automatically — installing packages and writing the Claude config. When you see "Installation Complete!", it's done.
4
Restart Claude Desktop
Fully quit Claude Desktop (right-click the taskbar icon → Quit), then reopen it. The plugin loads on startup.
✅
The installer automatically detects your Python location, installs all dependencies, and writes the Claude Desktop config file. No manual JSON editing needed.
⚠️
If the installer fails with a Python error, go back to Step 2 and make sure you ticked "Add Python to PATH" during Python installation.
5

Test the Connection

1
Open AutoCAD with any drawing
The drawing can be empty or an existing file — it doesn't matter for the connection test.
2
Open Claude Desktop — look for the MCP indicator
In Claude Desktop, start a new chat. Look for a small hammer/tool icon near the message box. If it shows "autocad" as a connected tool, the plugin is running.
3
Type this test prompt
Ask Claude to connect and report what it sees:
Connect to AutoCAD and tell me what drawing is currently open.
✅
If Claude responds with the name of your open drawing (e.g. "Drawing1.dwg is open"), the connection is working. You're ready to go!
ℹ️
If Claude says it cannot connect, make sure AutoCAD is open and fully loaded before asking Claude to connect. Also check Section 8 (Troubleshooting).

Your First Commands

Here are some ready-to-use prompts to get started. Copy and paste these into Claude Desktop with AutoCAD open.

A
Draw a simple floor plan
Connect to AutoCAD and draw a simple 10x8 metre floor plan with 3 rooms, add dimension lines and label each room.
B
Change text to a new coloured layer
Connect to AutoCAD, find all text in the drawing, create a new layer called TEXT_LABELS with red colour, and move all the text to that layer.
C
Design a house layout
Connect to AutoCAD and design a 1,000 sq.ft house with a living room, kitchen, 2 bedrooms, and a bathroom. Add walls and label the rooms.
D
List all layers in the drawing
Connect to AutoCAD and list all the layers in my current drawing with their colours and visibility status.
💡
Tip: Always start your prompt with "Connect to AutoCAD and..." — this ensures Claude establishes the connection before attempting any drawing commands.

Troubleshooting

!
Claude says it cannot find or connect to AutoCAD
Make sure AutoCAD is fully open with a drawing loaded before sending the prompt. If AutoCAD is still loading, wait for it to finish. Then restart Claude Desktop and try again.
!
The MCP tool icon does not appear in Claude Desktop
The installer did not run correctly or Claude was not restarted. Try these steps: (1) Run INSTALL.bat again as Administrator (right-click → Run as administrator). (2) Fully quit Claude Desktop — right-click the taskbar icon and choose Quit, then reopen it. (3) Start a new chat and check for the tool icon.
!
"pip install" fails with an error
Make sure you ticked "Add Python to PATH" during installation. If you missed it, uninstall Python and reinstall with that box checked. Then try pip install -r requirements.txt again.
!
Claude starts but stops mid-drawing
This is a usage limit on the free Claude plan. You've hit the daily message cap. Wait a few hours or upgrade to Claude Pro ($20/month) for higher limits. Your drawing up to that point is saved in AutoCAD.
!
Nothing happens in AutoCAD after Claude responds
Ensure AutoCAD is the active window when Claude issues commands, and check that the drawing is not locked or read-only. Try with a new blank drawing first.
